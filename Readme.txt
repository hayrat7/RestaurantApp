Mobile Appication built with flutter SDK and Spring Boot for Backend(Rest API).
the App shows a list of restuarants for clients by category,description screen for each restaurant using flutter hero animation,and position of restaurant in google map using google maps package.
an owner can add his restaurant,the app acces to his device's location using flutter location package,the owner can olso take a picture from the camera or add it from the gallery using flutter image picker package.
Authentifacation and register new users using Flutter Firebase auth package.
